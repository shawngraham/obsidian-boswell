# obsidian-boswell
a small plugin for project metadata tracking, project management

## Installation

### The harder way

- download this repository
- in your obsidian vault, find the hidden dot-folder `.obsidian` (in your finder or your file explorer, you have to reveal dot files)
- inside that folder, create a subfolder called `plugins`
- create a subfolder called `boswell`
- drag and drop the `manifest.json`, `main.js`, and `styless.css` files into `boswell`
- In obsidian, turn on community plugins
- Go to the community plugins panel in your settings
- Open the folder beside 'installed plugins'
- Find boswell.
- Select it

### The easier way
- turn on community plugins
- search for the 'BRAT' plugin
- install it
- Copy the link to boswell, ie `https://github.com/shawngraham/obsidian-boswell/`
- Open the command palette and run the command `BRAT: Add a beta plugin for testing`
- Using the github link for Boswell, copy that into the modal that opens up
- Click on Add Plugin -- wait a few seconds and BRAT will tell you what is going on
- After BRAT confirms the installation, in Settings go to the Community plugins tab.
- Refresh the list of plugins
- Find the beta plugin you just installed and Enable it.

The plugin gives you a couple of tools and modals to manage your humanities research project. See the documentation from the [webapp version](https://github.com/shawngraham/boswell) (the webapp version functions as a dashboard on top of a folder of local files that you point it at, so you can use it too with your obsidian vault if you don't want to faff about with all of this).
